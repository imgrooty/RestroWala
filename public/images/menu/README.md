# Menu Images Directory

This directory stores images for menu items and categories.

## File Organization

```
images/menu/
├── items/
│   ├── appetizers/
│   ├── main-course/
│   ├── desserts/
│   └── beverages/
└── categories/
    ├── appetizers.jpg
    ├── main-course.jpg
    ├── desserts.jpg
    └── beverages.jpg
```

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: Max 2MB per image
- **Dimensions**: 800x600px (4:3 ratio) recommended
- **Quality**: High quality for menu items
- **Background**: Clean, neutral backgrounds preferred

## Naming Convention

Use lowercase with hyphens:
- Good: `chicken-tikka-masala.jpg`
- Bad: `Chicken_Tikka_Masala.JPG`
